# Spec-Trace Spec Suite

This folder contains the canonical SPEC suite for the `spec-trace` standard.

The suite intentionally uses the standard to specify itself. It is the proving ground for the compact requirement-clause model.

The specification files include the full specification ID in the filename.

Canonical files:

- [SPEC-STD-0001](SPEC-STD-0001.md) - core definitions, authority model, and publication rules
- [SPEC-ID-0001](SPEC-ID-0001.md) - identifier policy and grouping-key registry
- [SPEC-LAY-0001](SPEC-LAY-0001.md) - repository layout and artifact placement
- [SPEC-TPL-0001](SPEC-TPL-0001.md) - templates and compact requirement grammar
- [SPEC-SCH-0001](SPEC-SCH-0001.md) - schemas and validation rules
- [SPEC-EXM-0001](SPEC-EXM-0001.md) - worked examples and traceability coverage

Root files such as `README.md`, `overview.md`, `layout.md`, and the templates summarize this suite but do not override it.
